# Social Cops News

Please add your api key to gradle.properties as : NewsApi_ApiKey="YOUR_API_KEY"

#### Libraries
- [Google gson](https://github.com/google/gson) For parsing JSON data
- [Retrofit](http://square.github.io/retrofit/) For sending network requests
- [Glide](https://github.com/bumptech/glide) Loading and caching images
- [Room, LiveData and ViewModel](https://developer.android.com/topic/libraries/architecture/)
